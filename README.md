# networking_dns_foundation

Scripts voor het ophalen en analyseren van DNS-informatie, geschreven in Python en PowerShell.

## Inhoud

| Map | Beschrijving |
|---|---|
| `py_app/` | Python-applicatie voor DNS-opvragingen |
| `py_conf/` | Configuratiebestanden voor de Python-applicatie |
| `ps1_app/` | PowerShell-scripts voor DNS-opvragingen |

> Elke submap bevat een eigen README met verdere uitleg.

## Vereisten

- Python 3.x
- PowerShell 5.1 of nieuwer

## Gebruik

Zie de README in de betreffende submap (`py_app/`, `py_conf/` of `ps1_app/`) voor instructies per onderdeel.
